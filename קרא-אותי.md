# לימוד קליל 🎓

עוזר לימודים אישי שרץ על המחשב שלך, מופעל על ידי Claude.

## איך מפעילים

1. **לחיצה כפולה על הקיצור "לימוד קליל" בשולחן העבודה** (או על `start.bat` בתיקייה) — נפתח חלון תוכנה.
2. בפעם הראשונה בלבד: **⚙ הגדרות** ← **🔑 התחבר למנוי** — נפתח חלון, בוחרים
   **Claude account with subscription** ומאשרים בדפדפן. זהו — התוכנה עובדת על מנוי ה-Pro/Max שלך, בלי תשלום נוסף.
   - (אפשרות חלופית: מפתח API מ-platform.claude.com, בתשלום לפי שימוש.)
3. צרו קורס, העלו את חומרי הלימוד (PDF / Word / PowerPoint / טקסט) — ומתחילים.

## מה יש בפנים

| לשונית | מה היא עושה |
|---|---|
| 📚 חומרים | העלאת סיכומי הרצאות, מצגות ודפי נוסחאות — הבסיס לכל השאר |
| 📝 סיכומים | סיכום מלא ומסודר של החומר, עם נקודות מפתח וטעויות נפוצות |
| ✏️ פתרון מטלות | פתרון צעד-אחר-צעד עם הסברים, בדיקת שפיות, ומה צפוי במבחן |
| 🎯 מבחני תרגול | מבחן ברמת סוף סמסטר — עונים, ומקבלים ציון ומשוב על כל שאלה |
| 🃏 כרטיסיות | כרטיסיות זיכרון לשינון מהיר, עם ערבוב |
| 💬 מורה פרטי | צ'אט שמכיר את חומרי הקורס ועונה על כל שאלה |

## טיפים לסטודנט מצטיין

- ככל שתעלו יותר חומר לקורס — הסיכומים, המבחנים והתשובות יהיו מדויקים יותר ובסגנון של הקורס שלכם.
- לפני מבחן: צרו מבחן תרגול, פתרו אותו בלי להציץ, ולחצו "בדוק אותי". המשוב מראה בדיוק איפה אתם חלשים.
- המודל המומלץ הוא Opus 4.8 (הכי חכם). אם רוצים לחסוך בעלויות — Sonnet 4.6 בהגדרות.

## איפה הכול נשמר

כל הקורסים, הסיכומים והמבחנים נשמרים בתיקיית `data/` — מקומית, אצלך במחשב.
גיבוי = להעתיק את התיקייה.
