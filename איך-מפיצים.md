# איך מפיצים את "לימוד קליל" להורדה

## מה כבר מוכן

הקובץ **`dist\LimodKalil.exe`** (כ-32MB) הוא תוכנה עצמאית לחלוטין:
- לא דורש פייתון ולא שום התקנה — מי שמוריד פשוט מריץ.
- מוגדר לעבוד עם **מפתח API** שכל משתמש מדביק בעצמו.
- שומר נתונים בתיקייה `LimodKalil-data` שנוצרת ליד הקובץ.

לצדו יש את `dist\קרא-אותי-לפני-שמתחילים.txt` — הוראות למי שמוריד.

## איך לבנות מחדש (אחרי שינויים בקוד)

```
python3.11 -m PyInstaller --noconfirm --clean LimodKalil.spec
```
הקובץ המעודכן ייווצר ב-`dist\LimodKalil.exe`.

## איך מפיצים — 3 דרכים

### 1. הכי פשוט: שיתוף קובץ
דחסו את `LimodKalil.exe` + קובץ ההוראות ל-ZIP, ושתפו ב-Google Drive / WeTransfer / מייל.
(אי אפשר לשלוח .exe ישירות במייל — Gmail חוסם; דחסו ל-ZIP או העלו לדרייב.)

### 2. GitHub Releases (מומלץ — חינמי ומקצועי)
1. פתחו חשבון ב-github.com וצרו Repository חדש.
2. בעמוד הריפו: Releases ← Create a new release.
3. גררו את `LimodKalil.exe` לאזור הקבצים, תנו שם וגרסה (למשל v1.0), ולחצו Publish.
4. תקבלו קישור הורדה ישיר לשיתוף.

### 3. אתר משלכם / itch.io
פלטפורמות כמו itch.io מאפשרות להעלות תוכנה חינמית או בתשלום עם עמוד הורדה יפה.

## שני דברים שחשוב לדעת

1. **אזהרת Windows (SmartScreen):** מכיוון שהקובץ לא חתום בתעודה דיגיטלית (שעולה ~$200-400 לשנה),
   Windows יציג "Windows protected your PC" בהפעלה הראשונה. זה לא וירוס — זה קורה לכל תוכנה
   חינמית לא חתומה. המשתמש לוחץ "More info" ← "Run anyway". כתבתי על זה בקובץ ההוראות.

2. **רק Windows:** הקובץ הזה עובד רק על Windows. לגרסת Mac צריך לבנות על מחשב Mac (אפשר, אבל דורש עבודה נפרדת).

## עדכון מהיר של ההוראות למשתמשים
הקובץ `dist\קרא-אותי-לפני-שמתחילים.txt` — ערכו אותו אם תרצו לשנות את ההסבר למורידים.
